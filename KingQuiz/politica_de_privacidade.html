🏰 O QuizLand respeita a sua privacidade. O app é um jogo RPG de perguntas e respostas que não coleta dados pessoais sem consentimento. O progresso e XP são salvos na nuvem somente com seu consentimento, via login com Google. 🏰 QuizLand respects your privacy. The app is an RPG quiz game that does not collect personal data without consent. Progress and XP are saved to the cloud only with your consent, via Google Sign-In.
📋

1. Informações Coletadas

1. Information Collected

O QuizLand não coleta nenhuma informação pessoal sem o seu consentimento. As configurações do jogo (nome, país, XP, nível, conquistas) são armazenadas apenas localmente no seu dispositivo via SharedPreferences.

QuizLand does not collect any personal information without your consent. Game settings (name, country, XP, level, achievements) are stored only locally on your device via SharedPreferences.

Caso o usuário opte por fazer login com o Google, o nome de exibição, foto de perfil, XP e conquistas são sincronizados com o Firebase Realtime Database para participação no ranking global.

If the user chooses to sign in with Google, the display name, profile photo, XP, and achievements are synced to Firebase Realtime Database to participate in the global leaderboard.

🗺️

2. Banco de Perguntas

2. Question Database

O QuizLand utiliza bancos de dados locais (.db SQLite) armazenados nos assets do aplicativo para exibir as perguntas dos quizzes. Esses arquivos são apenas de leitura e não contêm nenhum dado pessoal do usuário.

QuizLand uses local databases (.db SQLite) stored in the app assets to display quiz questions. These files are read-only and contain no personal user data.

O histórico de perguntas já respondidas é salvo localmente para evitar repetições, usando apenas IDs numéricos anônimos.

The history of answered questions is saved locally to avoid repetitions, using only anonymous numeric IDs.

🏆

3. Login com Google e Ranking Global

3. Google Sign-In and Global Leaderboard

O login com Google é completamente opcional. Sem ele, o jogo funciona normalmente e o progresso é salvo apenas no dispositivo.

Google Sign-In is completely optional. Without it, the game works normally and progress is saved only on the device.

Ao optar por fazer login, as seguintes informações são salvas no Firebase Realtime Database para exibição no ranking:

By choosing to sign in, the following information is saved to Firebase Realtime Database for leaderboard display:

  • Nome de exibição da conta Google
  • Código do país selecionado nas configurações
  • XP total acumulado e nível do jogador
  • IDs das conquistas desbloqueadas
  • Google account display name
  • Country code selected in settings
  • Total accumulated XP and player level
  • IDs of unlocked achievements

Nenhuma senha, e-mail ou dado financeiro é coletado ou armazenado.

No password, email, or financial data is collected or stored.

📢

4. Anúncios e Plano Premium

4. Ads and Premium Plan

O QuizLand é gratuito e sustentado por anúncios (banner e intersticial) exibidos via Google AdMob durante o jogo para usuários do plano gratuito.

QuizLand is free and supported by ads (banner and interstitial) displayed via Google AdMob during gameplay for free plan users.

Ao assinar o plano Premium, todos os anúncios são removidos completamente e recursos exclusivos são desbloqueados.

By subscribing to the Premium plan, all ads are completely removed and exclusive features are unlocked.

O Google AdMob pode utilizar um identificador de publicidade do dispositivo (Ad ID) para exibir anúncios relevantes. Você pode redefinir ou desativar esse identificador em Configurações → Google → Anúncios.

Google AdMob may use the device's advertising identifier (Ad ID) to display relevant ads. You can reset or disable this identifier under Settings → Google → Ads.

🔐

5. Permissões do Aplicativo

5. App Permissions

O QuizLand solicita apenas as seguintes permissões:

QuizLand requests only the following permissions:

🌐 INTERNET 📣 AD_ID

INTERNET: Necessária para o ranking global (Firebase), anúncios (AdMob) e login com Google.

INTERNET: Required for the global leaderboard (Firebase), ads (AdMob), and Google Sign-In.

AD_ID: Identificador de publicidade usado pelo Google AdMob para exibir anúncios relevantes a usuários do plano gratuito.

AD_ID: Advertising identifier used by Google AdMob to display relevant ads to free plan users.

🤝

6. Compartilhamento de Dados

6. Data Sharing

O QuizLand utiliza os seguintes serviços de terceiros:

QuizLand uses the following third-party services:

  • Google Firebase Realtime Database: Armazenamento do ranking global e conquistas
  • Google Sign-In / Firebase Auth: Autenticação opcional para o ranking
  • Google AdMob: Exibição de anúncios para usuários do plano gratuito
  • Google Play Billing: Processamento seguro de assinaturas Premium
  • Google Firebase Realtime Database: Global leaderboard and achievements storage
  • Google Sign-In / Firebase Auth: Optional authentication for the leaderboard
  • Google AdMob: Ad display for free plan users
  • Google Play Billing: Secure Premium subscription processing

Nenhuma informação pessoal é vendida, alugada ou compartilhada com terceiros para fins comerciais.

No personal information is sold, rented, or shared with third parties for commercial purposes.
