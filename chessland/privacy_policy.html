Privacy Policy

Last updated: January 20, 2026

Summary: This chess application does not require user registration, does not provide login functionality, and does not collect personal data.

1. Information We Collect

We do not collect personal information such as name, email address, phone number, or any other personally identifiable data.

All matches are recorded and stored in an anonymous manner, with no association to the identity of the player.

2. Game Data (Anonymous)

To enable game features such as global rankings and statistics, the application may store anonymous game-related data, including:

  • Game results
  • Game moves
  • Date and duration of matches

This data is stored anonymously using Supabase and cannot be used to identify any individual user.

3. Internet Usage

The application requires internet access in order to:

  • Save anonymous matches
  • Load global rankings
  • Synchronize game data

4. Data Sharing

We do not share any collected data with third parties for advertising, marketing, or tracking purposes.

5. Cookies

This application does not use cookies or similar tracking technologies.

6. Children's Privacy

This application is not specifically directed at children, and we do not knowingly collect personal data from children.

7. Security

We apply reasonable security measures to protect the anonymous data stored, including secure connections and reliable infrastructure.

8. Costs

The application is provided free of charge, with no mandatory purchases.

9. Changes to This Privacy Policy

If this Privacy Policy is updated, the revised version will be published on this page.
