Efeito Chroma Key no CapCut

Atualizado em: 19/03/2025 Tempo de leitura: 8 minutos CapCut, Chroma Key, Fundo Verde

Domine o efeito Chroma Key no CapCut e aprenda a remover fundos verdes (ou de outras cores) para substituí-los por qualquer imagem ou vídeo que você desejar, criando cenas profissionais como em filmes e programas de TV.

Dica: Assista ao vídeo tutorial abaixo para ver o efeito em ação antes de começar.
Demonstração Interativa: Passe o mouse sobre a área abaixo para ver a transformação do Chroma Key!
Antes: Com fundo verde
Depois: Com novo fundo

Pré-requisitos

  • CapCut versão mais recente instalada
  • Vídeo gravado com fundo verde (ou outra cor sólida)
  • Imagem ou vídeo para usar como novo fundo

Passo 1: Preparação do Vídeo

  1. Grave com fundo verde uniforme e bem iluminado
  2. Evite roupas ou objetos com cores próximas ao verde
  3. Mantenha boa distância entre pessoa/objeto e o fundo
  4. Use iluminação difusa para evitar sombras

Configuração ideal para gravação

Passo 2: Aplicando o Chroma Key

  1. Importe o vídeo com fundo verde para o CapCut
  2. Selecione o clipe na linha do tempo
  3. Toque em "Efeitos" > "Chroma Key"
  4. Use o conta-gotas para selecionar a cor do fundo
  5. Ajuste a intensidade e tolerância

Ferramenta Chroma Key no CapCut

Passo 3: Refinando o Efeito

  1. Ajuste o "Limiar" para remover todo o fundo
  2. Use "Suavização" para bordas mais naturais
  3. Compense sombras com "Compensação de Cor"
  4. Verifique frame a frame para ajustes precisos

Ajustes avançados do Chroma Key

Passo 4: Adicionando Novo Fundo

  1. Importe a imagem ou vídeo para o novo fundo
  2. Arraste para a linha do tempo ABAIXO do vídeo principal
  3. Ajuste duração e posicionamento
  4. Use "Modo de Mesclagem" se necessário

Adicionando novo plano de fundo

Dicas Avançadas

Iluminação

Grave com iluminação uniforme no fundo verde para melhores resultados.

Cores Alternativas

O CapCut permite usar outras cores além do verde para o chroma key.

Correção de Cor

Use a correção de cor para harmonizar o objeto com o novo fundo.

Solução de Problemas Comuns

Ajuste o limiar e use o conta-gotas novamente em áreas problemáticas. Verifique se a iluminação do fundo estava uniforme na gravação.

Aumente a suavização e reduza levemente o limiar. Gravar em resolução mais alta também ajuda.

Diminua o limiar e verifique se não há cores similares ao fundo no objeto. Use a compensação de cor se necessário.
